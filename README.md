## 介绍
* 本项目是xray洞鉴漏扫工具的辅助插件，主要解决使用过程中的一些小问题，主要功能如下
1. 导出所有漏洞到文本，某些时候需要写一些报告要用到这些数据
2. 解决X-Ray扫描结果web站点，点击总是进入详情的问题（大多数时候我们想用浏览器访问这个链接，而不是进入详情），新增浏览器打开按钮
3. 新增批量导出Web站点url的功能，某些时候需要批量处理这里url
